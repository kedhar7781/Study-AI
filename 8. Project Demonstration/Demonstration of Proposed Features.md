# Phase 8: Demonstration of Proposed Features

This document provides a step-by-step feature demonstration script to showcase **StudyAI** during demo day.

---

## 1. Feature Demonstration Checklist

Use this script to walk through the visual and functional capabilities of the system:

### Section 1: Landing Page & Onboarding
- [ ] Show the marketing hero banner, feature grid, and accordion FAQs.
- [ ] Sign up as a new user, noting the password validation and JWT cookie setting.
- [ ] Show the dark/light theme toggle.

### Section 2: Document Ingestion & Summary Generation
- [ ] Upload a sample PDF or TXT syllabus document.
- [ ] Open the material list, select the document, and click "Generate Summary".
- [ ] Review the markdown summary output (definitions, takeaways, memory tricks).
- [ ] Copy the raw summary markdown and download the compiled summary PDF.

### Section 3: Active recall Practice
- [ ] Open the Flashcards workspace, generate a deck, and click cards to flip them.
- [ ] Rate card difficulties (easy, medium, hard) and toggle favorite stars.
- [ ] Open the Quiz Practice hub, generate a 5-question MCQ test.
- [ ] Play through the quiz, submit answers, and view the Recharts score pie charts.
- [ ] Review correct answers and explanation cards.

### Section 4: Calendar study planning
- [ ] Open the Planner workspace and generate a 7-day schedule sprint.
- [ ] Check off daily checklists.
- [ ] Open the Analytics page and view area chart progressions.
